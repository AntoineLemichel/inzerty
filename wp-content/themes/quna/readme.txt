== Description ==
Minimal WordPress theme with modern and clean design, which supports Gutenberg editor. The main purpose is to provide homepage/landing page layout that is purely composed with Gutenberg blocks. To allow richer blocks, Quna is compatible with Atomic Blocks. It also has fabulous posts archive page. In addition, it allows to display social media icons links and latest photos from Instagram feed. This theme is best for personal blog and small business; as well as specific purposes like restaurant/cafe, personal trainer, or photography.
